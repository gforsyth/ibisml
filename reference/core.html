Recipe

Recipe(self, *steps)

A recipe for fitting a preprocessing transform.

Recipes combine one or more preprocessing steps, applied in series.

TransformResult

TransformResult(self, table, features=None, outcomes=None, other=None, categories=None)

The result of applying a RecipeTransform.

This is a wrapper around an ibis Table containing an expression describing the full preprocessing pipeline. Use one of the to_* methods to execute the expression and convert the result into an input type usable for fitting your model.
